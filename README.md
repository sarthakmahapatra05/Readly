## Overview

**Readly** is a modern, community-driven platform that transforms how people discover and consume knowledge through concise book summaries.

The platform enables users to:
- Explore high-quality summaries
- Share their own insights
- Build personalized reading collections
- Connect with a growing learning community

Readly is designed for students, professionals, and book enthusiasts who want to learn faster and smarter.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/sarthakmahapatra05/Readly.git
cd Readly
```

---

### Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

---

### Configure Environment Variables

Create a `.env.local` file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

---

### Start Development Server

```bash
npm run dev
```

or

```bash
yarn dev
```

---

### Open in Browser

```bash
http://localhost:8080
```
